fvAvatar

To create user avatars use this component.


Code:

  

    HTML
    
  

  <!-- Normal -->
<fvAvatar src="path/to/avatar.jpg" name="Amir Hashemi" />

<!-- Character Mode -->
<fvAvatar name="Amir Hashemi" />
