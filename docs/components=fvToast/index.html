fvToast

To show toast message (alert, custom messages) use this component.


Code:

  

    HTML
    
  

  <fvMain parent>
  <fvToast v-model="toast" :timeout="3500">
    <!-- Toast Contents -->
  </fvToast>
</fvMain>
  

  

    Javascript
    
  

  export default {
    data () {
        return {
            toast: false
        }
    }
}

Notes:

  • fvToast should atleast one fvMain parent with parent prop.
