Dropdown Select

GITHUB

To create full featured select (dropdown) element use this component.


Code:

  

    HTML
    
  

  <fvSelect v-model="input" :options="options" />
  

  

    Javascript
    
  

  export default {
    data () {
        return {
            input: undefined,
            options: [
                {
                    text: 'Item One',
                    value: 1
                },
                {
                    text: 'Item Two',
                    value: 2
                },
                {
                    text: 'Item Three',
                    value: 3,
                    disabled: true
                }
            ]
        }
    }
}
