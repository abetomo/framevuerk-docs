Steps

GITHUB

To create step highlighter in top of your forms use this component.


Code:

  

    HTML
    
  

  <fvSteps :value="value" :steps="steps" />
  

  

    Javascript
    
  

  export default {
    data () {
        return {
            steps: ['Stage One', 'Stage Two', 'Stage Three', 'And Stage Four'],
            value: 0
        }
    }
}
